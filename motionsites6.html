1. Tóm tắt Ý tưởng Thiết kế (Design Concept)

Theme

"The Pulse of Intelligence" (Nhịp đập Trí tuệ). Một giao diện phản hồi theo thời gian thực.

Focus

Sự giao thoa giữa dữ liệu phức tạp và giao diện người dùng tối giản, tinh gọn.

Style

Luxury Minimalism: Sử dụng Glassmorphism, khoảng trắng và hoạt ảnh mượt mà.

2. Hệ thống Nhận diện Thị giác

Bảng màu (Color Palette)

Deep Black (#000000)

Màu chủ đạo, tạo chiều sâu và sự sang trọng.

Steel Grey (#505050)

Sử dụng cho các tiêu đề phụ và văn bản bổ trợ.

Off-White (#F8F8F8)

Nền cho các vùng sáng, đảm bảo khả năng đọc nội dung.

Pulse Green (rgba(90, 225, 76, 0.89))

Màu nhấn cho các nút hành động quan trọng (CTA).

Hệ thống Kiểu chữ (Typography)

Fustat Bold

Tiêu đề lớn, hình học hiện đại.

Schibsted Grotesk

Thanh điều hướng và UI elements.

Inter Regular

Nội dung chi tiết và nhãn thông tin.

5. Mã nguồn Hero Section (React Component)

HeroSection.tsx 
import React, { useEffect, useRef } from 'react';

// Custom High-Performance Fade System
const HeroSection = () => {
  const videoRef = useRef<HTMLVideoElement>(null);
  let fadeAnimFrame: number;

  const fade = (target: number, duration: number) => {
    const startOpacity = parseFloat(videoRef.current?.style.opacity || '0');
    const startTime = performance.now();

    const animate = (now: number) => {
      const elapsed = now - startTime;
      const progress = Math.min(elapsed / duration, 1);
      if (videoRef.current) {
        videoRef.current.style.opacity = (startOpacity + (target - startOpacity) * progress).toString();
      }
      if (progress < 1) fadeAnimFrame = requestAnimationFrame(animate);
    };
    fadeAnimFrame = requestAnimationFrame(animate);
  };

  return (
    <div className="relative h-screen overflow-hidden bg-black">
      <video 
        ref={videoRef}
        className="absolute min-w-[115%] min-h-[115%] object-cover object-top opacity-0"
        autoPlay muted
      >
        <source src="/path-to-ai-motion.mp4" />
      </video>
      {/* Content Layers... */}
    </div>
  );
};
